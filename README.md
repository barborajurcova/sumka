# 📱 Sumka - Pokladna do kapsy

**Sumka** je moderní, bleskurychlý pokladní systém navržený speciálně pro **stánkový prodej**, trhy a malé provozovny. Zapomeňte na složité papírování a drahé pokladní terminály – Sumka vám dává kompletní přehled o tržbách přímo ve vašem mobilu nebo tabletu.

![Sumka Banner](https://imgur.com/OoIfcZc.png)

## ✨ Hlavní funkce

* ⚡ **Bleskový prodej:** Markování produktů na dvě kliknutí – ideální pro rušné stánky s frontami.
* 💸 **Chytré slevy:** Okamžité aplikování procentuálních slev bez nutnosti počítání z hlavy.
* 📊 **Přehledy v reálném čase:** Sledujte tržby rozdělené na hotovost a karty během celého dne.
* 📂 **Export pro účetní:** Generování denních uzávěrek v CSV formátu, které stačí jen přeposlat.
* 📱 **Mobile-first:** Rozhraní optimalizované pro ovládání jednou rukou na telefonu.
* 📦 **Správa sortimentu:** Snadná úprava cen a názvů produktů přímo v terénu.

## 🛠️ Technologie

Projekt je postaven na moderním webovém stacku pro maximální rychlost a stabilitu:

* **React** (v18+)
* **Vite** (Build tool)
* **Tailwind CSS** (Styling)
* **Lucide React** (Ikony)
* **Supabase** (Backend & Databáze) 

## 🚀 Jak začít (Instalace u sebe)

Pokud si chcete projekt spustit lokálně pro vývoj, postupujte podle těchto kroků:

## 1. Klonování repozitáře
Nejprve si stáhněte kód k sobě do počítače:
```bash
git clone [https://github.com/barborajurcova/sumka.git](https://github.com/barborajurcova/sumka.git)
cd sumka

## 2. Instalace závislostí

Protože složka node_modules se na GitHub nenahrává, musíte si všechny potřebné knihovny nainstalovat:

Bash
npm install

## 3. Spuštění vývojového serveru

Nyní můžete aplikaci spustit:

Bash
npm run dev

Aplikace poběží standardně na adrese http://localhost:5173.

## Nasazení (Deployment)
Pro produkční sestavení použijte příkaz:

Bash
npm run build

Výslednou složku dist pak můžete nahrát na jakýkoliv hosting (Vercel, Netlify, atd.).
